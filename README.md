## Project Installation

* PHP dependencies
  
```bash
composer install
```

* JS dependencies
  
```bash
npm install
```

* Run PHP code locally
Copy .env.example to .env, configure your environment variables and execute:

```bash
php artisan key:generate
```

```bash
php artisan serve
```

* Run JS code
```bash
npm run dev
```
